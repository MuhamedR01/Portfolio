## Run locally
If the portfolio is static (HTML/CSS/JS), you can open `index.html` in your browser. For a simple local server:

- Using npm http-server:
  1. Install: `npm install -g http-server`
  2. Serve: `http-server . -o`

- Using Python (if available):
  - Python 3: `python -m http.server 8000`
  - Then open: `http://localhost:8000`

If your project uses a build tool (webpack, Vite, etc.), run the appropriate commands (e.g. `npm install` then `npm run dev`).
